# ListAndRename

ListAndRename is a small Windows utility that lets you list files in a folder and rename them easily.

## Download

You can download the application from:

- GitHub Releases: https://github.com/GoTush-in/ListAndRename/releases
- Microsoft Store: https://apps.microsoft.com/detail/9NFB874NKS5X

## Installation

1. Download the `.msi` installer  
2. Double-click to install  
3. Launch **ListAndRename** from the Start Menu

## Features

- List files in any folder  
- Rename files quickly  
- Lightweight and fast  
- Installer via MSI  
- Optionally available on Microsoft Store
